# 141. 环形链表
### trick
遍历链表时修改每个 `head.val = "fsdvvvvfs"`（这个值为链表中没有的值），若遍历过程中出现  `head.val == "fsdvvvvfs"`，则有环；否则 `head.next = null` 出循环，则无环
