本题中的情况：若出现 5 个 n，那么每次都是从第 n 位开始往后探测，显然会很耗时。所以每次放置好一个 n 后，把 `pos[n]` 修改为新的位置。比如第 4 个 n 放在 n + 3 的位置，那么放置好之后就把 `pos[n] = n + 3`，那么下次探测就从 `n + 3` 开始
